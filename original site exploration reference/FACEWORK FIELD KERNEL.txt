II. THE FIELD MODEL
We model identity as a continuous scalar field
generated by the superposition of waves emitted from multiple sources.
Let (x, y) \in [-1,1]^2.
Let t \ge 0 be time.
Let each source i have:
* amplitude A_i
* wavelength \lambda_i
* angular frequency \omega_i
* center c_i = (c^x_i, c^y_i)
* base phase \phi_i^0
* decay radius \sigma_i
* drift rate \alpha_i
Distance to source:
r_i = \sqrt{(x' - c^x_i)^2 + (y' - c^y_i)^2}
with drifted coordinates:
x' = x + v_x t,\quad y' = y + v_y t
Contribution from source i:
f_i(x,y,t) = A_i \cdot \cos\left( \frac{2\pi}{\lambda_i} r_i - \omega_i t + \phi_i(t) \right) \cdot e^{-r_i^2 / (2\sigma_i^2)}
Phase drift:
\phi_i(t) = \phi_i^0 + \alpha_i t
Total field:
F(x,y,t) = \sum_{i=1}^N f_i(x,y,t)
This raw field is then shaped by the primitives through coherence-regulated transformations.

1) Purpose

It answers:
“How fast can this system move while still staying coherent?”
This is the threshold engine:
one notch too far and you fall into Diagnostic collapse.
2) Mathematical definition (speed-applied kernel)

F(x,y,t)=\sum_{i=1}^{N}A_i\cos\left(\frac{2\pi}{\lambda_i}r_i-\omega_i t+\phi_i(t)\right)e^{-r_i^2/(2\sigma_i^2)}

Entropy exists but is clamped by coherence:

So Velocity is not high entropy; it’s high speed.

ENGINE-TO-ENGINE TRANSITION RULES
0) Setup: what is transitioning?
Each engine is a parameter set:
\theta = \{C,\ \lambda_0,\ \sigma_\lambda,\ \omega_0,\ \sigma_\omega,\ v_x,\ v_y,\ N,\ \phi\text{-spread},\ \alpha,\ S,\ g_d,\ E\}
A transition from engine A to B is:
\theta(t) = \text{Interp}(\theta_A,\theta_B,\ u(t))
where u(t)\in[0,1] is a time-normalized progress variable.
1) Standard interpolation primitives
1.1 Smoothstep for continuous params
Use cubic smoothstep (stable, no overshoot):
s(u)=u^2(3-2u)
Then:
p(u)=p_A+(p_B-p_A)\cdot s(u)
Use this for:
C, λ0, σλ, ω0, σω, vx, vy, phaseSpread, α, S, E
1.2 Integer parameters (sources N, gridDensity gd)
Do thresholded step at defined breakpoints, not linear interpolation.
Let u\in[0,1].
                                                                                                                                                                                                                           * For gridDensity:
                                                                                                                                                                                                                           * switch at u=0.65
                                                                                                                                                                                                                           * For source count N:
                                                                                                                                                                                                                           * switch at u=0.55
So:
g_d(u)=\begin{cases} g_{dA} & u<0.65\\ g_{dB} & u\ge0.65 \end{cases} \quad N(u)=\begin{cases} N_A & u<0.55\\ N_B & u\ge0.55 \end{cases}
Rationale: changing these too early causes topology jumps.
1.3 Phase seed continuity (prevents “teleport”)
Keep existing phases \phi_i^0 and blend the spread, not the phase values.
Maintain phase ordering; only change the spread scalar.
If you store a normalized phase vector q_i\in[-1,1] per source, then:
\phi_i^0(u)=\bar\phi + q_i\cdot \sigma_\phi(u)
where \sigma_\phi(u) is interpolated with smoothstep.
This prevents random re-seeding.
2) Global transition timeline (recommended)
Use a 3-phase schedule to avoid tearing:
Duration
                                                                                                                                                                                                                           * Default: 3.5s for “adjacent” engines
                                                                                                                                                                                                                           * Default: 5.0s for “non-adjacent” engines (Identity→Cultural, etc.)
Phases (normalized u)
                                                                                                                                                                                                                           1. Stabilize geometry: u\in[0,0.35]
Move Coherence + Stability first.
                                                                                                                                                                                                                           2. Shift dynamics: u\in[0.35,0.80]
Move Current + Flow + Frequency variance.
                                                                                                                                                                                                                           3. Reveal texture: u\in[0.80,1.00]
Move Entropy + Resonance (phase spread) last.
This ordering is crucial. If you move entropy early, everything tears.

const eB = smoothstep(remap(u, 0.35, 0.80));

// continuous params
C  = lerp(CA,  CB,  eA);
S  = lerp(SA,  SB,  eA);


vx = lerp(vxA, vxB, eB);

// late params
E  = lerp(EA,  EB,  eC);
phiSpread = lerp(phiA, phiB, eC);


// integer params (late step)
N  = u < 0.55 ? NA : NB;
gd = u < 0.65 ? gA : gB;


// clamp effective entropy
eps = E * (1 - C);
eps = clamp(eps, epsMin, epsMax);
E = eps / (1 - C);

1.2 Master field
We define the Facework Field as a coherence-regulated wave interference field:
\Psi(x,y,t)=\mathcal{N}\Big( \underbrace{\sum_{i=1}^{N} A_i \cos\big(k_i\,\|[x,y]-s_i(t)\|-\omega_i t+\phi_i(t)\big)}_{\text{interference core}} \cdot \underbrace{D(x,y)}_{\text{boundary/damping}} \cdot \underbrace{(1-S)+S\,M_{g_d}(x,y)}_{\text{stability mask blend}} \;+\; \underbrace{\epsilon\,\eta(x,y,t)}_{\text{entropy}} \Big)
Where:
Interference core
                                                                                                                                                                                                                                             * N = number of sources (Resonance)
                                                                                                                                                                                                                                             * A_i = amplitudes (Resonance weighting)
                                                                                                                                                                                                                                             * k_i = \frac{2\pi}{\lambda_i} (Frequency)
                                                                                                                                                                                                                                             * \omega_i = angular frequencies (Frequency)
                                                                                                                                                                                                                                             * s_i(t) = source positions (Current)
                                                                                                                                                                                                                                             * \phi_i(t) = phases (Flow)
Boundary/damping
                                                                                                                                                                                                                                             * D(x,y) keeps the field bounded (Stability foundation; formalized in step 3)

Stability mask
                                                                                                                                                                                                                                                * S\in[0,1] is Stability strength
                                                                                                                                                                                                                                                * M_{g_d}(x,y)\in\{0,1\} is a grid-like mask with density g_d
Entropy
                                                                                                                                                                                                                                                * \eta(x,y,t) is zero-mean noise (seeded, continuous in time)
                                                                                                                                                                                                                                                * \epsilon is effective entropy strength
Normalization
                                                                                                                                                                                                                                                * \mathcal{N}(\cdot) maps to a stable renderable range (e.g. [-1,1]) and prevents amplitude drift across engines
This is the full system. Everything else is parameterization.

DEFINE COHERENCE AS MATH
Coherence is not a vibe. It is a constraint operator that regulates variance and entropy in the master equation.
We define coherence C\in[0,1] as a set of deterministic mappings:
2.1 Coherence regulates variance (Frequency stability)
Let \lambda_i and \omega_i be drawn from base distributions:
\lambda_i \sim \mathcal{N}(\lambda_0,\sigma_\lambda^2),\quad \omega_i \sim \mathcal{N}(\omega_0,\sigma_\omega^2)
Coherence shrinks those spreads:
\sigma_\lambda^{\text{eff}} = \sigma_\lambda\,(1-C)
\sigma_\omega^{\text{eff}} = \sigma_\omega\,(1-C)
So when C\to 1, frequencies converge; when C\to 0, diversity explodes.
2.2 Coherence regulates phase dispersion (Resonance stability)
If phases are drawn from:
\phi_i^0 \sim \mathcal{N}(\bar\phi,\sigma_\phi^2)
Then:
\sigma_\phi^{\text{eff}}=\sigma_\phi(1-C)
2.3 Coherence clamps effective entropy (breakage control)
Define effective entropy used in the master equation:
\epsilon = E\,(1-C)
where E\in[0,1] is the Entropy primitive parameter.
This makes coherence the governing primitive:
                                                                                                                                                                                                                                                * high C suppresses noise
                                                                                                                                                                                                                                                * low C exposes noise
2.4 Coherence adds a structural normalization constraint (global stability)
To prevent amplitude explosion across engines, we define:
\mathcal{N}(u)=\frac{u}{\max(\delta, \mathrm{RMS}(u))}
computed over the render grid per frame (or over a fixed calibration set).
                                                                                                                                                                                                                                                * \delta is a small constant to prevent division by zero.
This is coherence’s “system holds” behavior: the field remains comparable across modes.
(If you want a stricter alternative, we can use tanh normalization: \mathcal{N}(u)=\tanh(\beta u).)

OBSERVABILITY (MEASUREMENT LAYER)
This is the layer that lets Facework know what it’s doing. These are computed directly from the rendered field \Psi(x,y,t) over the domain (x,y)\in[-1,1]^2.
Assume you sample \Psi on a grid and treat it as a matrix \Psi_{ij}.
We’ll define five observable metrics (minimum viable set), plus two optional ones that are worth it.
4.1 Core observables
O1 — Coherence Score K (0–1)
Coherence in practice = how structurally consistent the field is.
We measure it as “low spatial roughness” (smooth, continuous interference) vs noisy tearing.
Compute field gradients (finite differences):
\partial_x\Psi,\ \partial_y\Psi
Compute mean squared gradient energy:
G=\mathbb{E}\left[(\partial_x\Psi)^2+(\partial_y\Psi)^2\right]
Normalize against a kernel reference G_{\text{ref}} (a fixed calibration value measured from a mid-state baseline render):
K=\exp\left(-\frac{G}{G_{\text{ref}}}\right)
                                                                                                                                                                                                                                                      * K\approx 1: smooth, structurally coherent
                                                                                                                                                                                                                                                      * K\approx 0: jagged/noisy, incoherent
Locked requirement: you must set G_{\text{ref}} once from a reference field (Identity canonical config).
O2 — Entropy Index H (0–1)
Entropy is “how random the field is” independent of gradient.
We compute a histogram of \Psi values (e.g. 64 bins), probability mass p_b, then Shannon entropy:
H_{\text{raw}}=-\sum_b p_b \log p_b
Normalize by max entropy \log(B) for B bins:
H=\frac{H_{\text{raw}}}{\log(B)}
                                                                                                                                                                                                                                                      * High H: distribution looks like noise
                                                                                                                                                                                                                                                      * Low H: distribution is structured (even if complex)
This helps distinguish:
                                                                                                                                                                                                                                                      * “complex but coherent” vs “just noisy”.
O3 — Stability Index T (0–1)
Stability is about anchoring: does the field have persistent structure over time?
We measure temporal consistency via frame correlation.
Sample field at t and t-\Delta t_s where \Delta t_s is a fixed sampling window (e.g. 0.5s).
Compute Pearson correlation:
\rho=\text{corr}(\Psi(t),\Psi(t-\Delta t_s))
Map to 0–1:
T=\frac{\rho+1}{2}
                                                                                                                                                                                                                                                      * T\approx 1: highly stable (Infrastructure)
                                                                                                                                                                                                                                                      * T\approx 0.5: moderate evolution (Narrative/Cultural)
                                                                                                                                                                                                                                                      * T\approx 0: chaotic / tearing (Diagnostic)
Locked default: \Delta t_s=0.5 seconds.
O4 — Current Magnitude J (0–1)
Current is directional movement. We want a measured drift magnitude based on parameters, but also optionally from field motion.
Parameter-based (locked minimal):
J=\text{clamp}\left(\frac{\|\mathbf{v}\|}{v_{\max}},0,1\right)
Where:
                                                                                                                                                                                                                                                      * \|\mathbf{v}\|=\sqrt{v_x^2+v_y^2}
                                                                                                                                                                                                                                                      * v_{\max} is a kernel constant.
Locked default: v_{\max}=0.45 (the top of Velocity range).
Optional field-based later: optical flow; not required yet.
O5 — Resonance / Complexity Index R (0–1)
Resonance should read as “how richly the interference interacts.”
We measure it as spectral richness: how many spatial frequencies are present.
Compute 2D FFT magnitude |\hat\Psi|. Let P be normalized power spectrum.
Compute spectral entropy:
R_{\text{raw}}=-\sum_k P_k\log P_k
Normalize:
R=\frac{R_{\text{raw}}}{\log(N_k)}
                                                                                                                                                                                                                                                      * Low R: simple pattern (few modes)
                                                                                                                                                                                                                                                      * High R: rich interference (Cultural)
This is the “how many layers are active” metric.
4.2 Optional observables (worth adding)
O6 — Symmetry Score Y (0–1)
Identity/Infrastructure should have stronger symmetry than Diagnostic.
Define rotated/mirrored comparisons (choose one):
                                                                                                                                                                                                                                                      * horizontal mirror: \Psi(x,y) vs \Psi(-x,y)
                                                                                                                                                                                                                                                      * vertical mirror: \Psi(x,y) vs \Psi(x,-y)
                                                                                                                                                                                                                                                      * 180° rotation: \Psi(x,y) vs \Psi(-x,-y)
Compute correlation for each and average:
Y=\text{mean}\Big(\frac{\text{corr}(\Psi,\Psi_{\text{mirror}})+1}{2}\Big)
O7 — Fracture / Tear Index F (0–1)
Diagnostic needs fracture localization, not uniform noise.
Compute gradient magnitude map:
g(x,y)=\sqrt{(\partial_x\Psi)^2+(\partial_y\Psi)^2}
Compute the fraction of pixels above a high threshold (e.g. 95th percentile of baseline):
F=\Pr(g>g_{95,\text{ref}})
                                                                                                                                                                                                                                                      * High F: lots of sharp discontinuities/edges
                                                                                                                                                                                                                                                      * Low F: continuous field
