# Examples

Reference output from completed Facework protocol runs.

These directories contain the artifacts produced by running the protocol on
real projects. They exist as reference material — to show what each phase
produces and what a complete run looks like.

**These are NOT the active project.** If you are running the protocol on a
new project, ignore this directory entirely. Your output will be created in
a new project-specific workspace.

## face.works/

The first protocol run, applied to Facework's own commercial infrastructure.

- `define/` — Governance, business model, decisions, playbooks, gate structure
- `prototype/` — Next.js site (the face.works product prototype)

This run produced: 16 locked decisions, 5 operational playbooks, a conformance
model, engagement delivery spec, business model, and a working prototype.
