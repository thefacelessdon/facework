## Stage Boundaries (Locked)

### MVP — Services Phase
**Entry:** Now (Frequency extracted, Current locked)
**Exit criteria (compound gate — BOTH required):**
1. Revenue: $5K+/month for 3 consecutive months
2. Proof: 3 completed protocol runs with documented outcomes, 1+ public reference

**What MVP is:** You run the protocol directly for creators/cultural brands.
**What MVP is not:** Licensing, certification, studio partnerships, or tooling sales.

### Beta — Transfer Phase
**Entry:** MVP compound gate cleared
**Exit criteria:**
1. At least 1 studio successfully running the protocol under license
2. Licensed studio produces a conformant system without founder involvement
3. Revenue mix includes both services AND licensing

**What Beta is:** The protocol is transferable. Others can run it.
**Identity shift:** Facework goes from "you running it" to "the protocol others run."

### Scale — Standard Phase
**Entry:** Beta exit criteria met + minimum 5 completed protocol runs total
**Exit criteria:**
1. Certification program launched and operational
2. Multiple studios licensed
3. "Facework Protocol Conformant" has market recognition

**What Scale is:** The protocol is a recognized standard with an ecosystem.

## No-Go Lines (Locked)

**Hard kill triggers:**
- 12 months from first payment received, revenue below $5K/month for 3+
  consecutive months → restructure commercial model
- A non-negotiable is violated and cannot be restored → stop and fix
- Author is sole source of protocol credibility after 12 months → extraction,
  must fix

**Soft pivot triggers (investigate, don't kill):**
- 6 months with no paid engagement → reassess go-to-market
- First 3 engagements produce no referrals → output quality or market fit issue
- A creator engagement fails the extraction check → methodology gap

## Non-Negotiables (Locked)

These cannot be traded away for speed, revenue, or convenience:

1. **Protocol stays open.** Theory, methodology, spec — always freely available.
2. **Creator sovereignty is absolute.** Every engagement passes the extraction
   check. Creator owns all output. Exit is clean.
3. **No lock-in.** Deliverables in open formats. No proprietary dependencies.
   Creator's system works without Facework.
4. **Conduits, not containers.** Facework never controls creator infrastructure,
   distribution, data, or economic current.
5. **Energy transfer within 12 months.** If the protocol still depends entirely
   on the author's energy after 12 months, it's an extraction pattern — fix it.
