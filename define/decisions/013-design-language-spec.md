# Decision 013: Design Language Spec Required in Resonance Phase

## The Question
Should every engagement produce a formal Design Language Spec, or is
design system work outside the protocol's scope?

## Resolution
Design Language Spec is a required artifact for Full Protocol engagements
(produced during the Resonance phase). It covers typography, color (mapped
to states), grid + layout, motion principles, iconography, and tone.

The spec is derived FROM the creator's frequency, not imposed on it.
Facework's own Visual Language System (in `original site exploration
reference/`) is the reference implementation.

## Alternatives Considered
- Recommended but not required: Rejected — if the spec is optional,
  prototypes will default to generic SaaS patterns. The whole point
  of Resonance is that the output carries the frequency.
- Facework-specific only: Rejected — the problem isn't unique to
  Facework. Every creator needs their visual language to be structural,
  not decorative. Making it protocol-level prevents the need for
  random design skills that bring work out of coherence.

## Reasoning
The Five Laws check (energy moves, emotion leads form, rhythm over rules,
friction reveals imbalance, dual worlds one current) evaluates whether a
prototype is alive. The Design Language Spec ensures the visual system is
*coherent* — that colors mean things, motion communicates state, and
typography carries identity. Without it, prototypes pass the Five Laws
but fail the frequency test.

## Implications
- Engagement delivery spec updated: Design Language Spec added to
  Phase 5 (Resonance) deliverables
- Conformance model should check for Design Language Spec at Level 3+
- Foundation Only engagements skip this (no Resonance phase)
- The spec structure (typography, color, grid, motion, iconography, tone)
  is consistent across engagements but the *content* varies per creator

## Coherence Impact
High coherence. The Design Language Spec is the visual implementation of
Frequency. Without it, the prototype's visual identity is disconnected
from the governing truth established in Phase 1.
