# /fw-evolve

Close the loop by converting execution evidence into methodology updates.

## Sequence
1. Run `/weekly-upgrade` for immediate system improvements.
2. Run `/fw-diagnostic` for full primitive and cultural coherence retrospective.
3. Update methodology docs and changelog with approved improvements.
4. Publish next-cycle operating guidance.

## Output
- `Retro findings`
- `Method updates`
- `Version/changelog updates`
- `Next-cycle directives`
