# /fw-product-wedge

Translate validated discovery into a coherent MVP wedge and execution path.

## Sequence
1. Use outputs from `/fw-discovery-loop`.
2. Run `/mvp-cut` for smallest high-integrity build slice.
3. Run `/fw-current` to resolve strategic dilemmas before implementation.
4. Run `/fw-stability` to define architecture and acceptance criteria.

## Output
- `Wedge definition`
- `Decision lock summary`
- `Architecture-ready handoff`
