# /fw-discovery-loop

Run the front-end of the cultural loop from hunch to grounded signal.

## Sequence
1. Run `/fw-semantics` to extract meaning and audience interpretation boundaries.
2. Run `/fw-field` to model norms, incentives, adoption dynamics, and extraction vectors.
3. Run `/fw-frequency` to lock governing economics and ownership boundaries.
4. Output a discovery brief ready for wedge design.

## Output
- `Discovery brief`
- `Signal and field summary`
- `Governing truth lock`
- `Open risks before wedge`
