# /fw-launch-loop

Prepare and execute launch with coherence and reliability checks.

## Sequence
1. Run `/beta-hardening` to close reliability gaps.
2. Run `/fw-consonance` to verify cross-layer alignment.
3. Run `/launch-ops` for release execution and post-release validation.
4. Capture launch outcomes and unresolved risk items.

## Output
- `Launch readiness`
- `Consonance check result`
- `Release and post-release report`
