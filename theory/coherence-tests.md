# Coherence Tests

Coherence must be testable, not assumed.

## Required Test Set

1. **Signal Test**  
   The intended cultural signal is legible to the communities it claims to serve.

2. **Decision Test**  
   Strategic and technical decisions do not contradict each other downstream.

3. **Architecture Test**  
   Implementation constraints encode ownership and governance choices explicitly.

4. **Behavior Test**  
   Operational playbooks reinforce declared values in real edge-case conditions.

5. **Transmission Test**  
   Product artifacts transmit meaning without the original author in the loop.

6. **Handoff Test**  
   A new operator can execute safely using only the documented system.

7. **Extraction Test**  
   Scaling does not increase value capture asymmetry against source communities.

## Failure Protocol

If any test fails:
- freeze expansion scope,
- run an entropy pass on the failing layer,
- document corrective action and re-test before proceeding.
